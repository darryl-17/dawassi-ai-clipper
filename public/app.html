Publish to YouTube

Your channel
Connected — hit Publish on any clip below to upload it in one click.
Uploads use the YouTube Data API (~1,600 quota units each; ~6 uploads/day on the default quota). If your Google project hasn’t been verified by YouTube yet, uploads may be forced to Private until you request an API audit.
Your Google credentials are saved. Connect your channel to authorise uploads.
To publish to your own channel, YouTube requires a free Google OAuth app (a one-time, ~10-minute setup). Paste the Client ID & Secret you create below.
http://localhost:3547/api/youtube/callback
Step-by-step: create your Google credentials
  1. Go to Google Cloud Console and create a new project.
  2. In APIs & Services → Library, enable the YouTube Data API v3.
  3. In OAuth consent screen, choose External, fill the basics, and add your own Google account under Test users.
  4. In Credentials → Create credentials → OAuth client ID, pick Web application.
  5. Under Authorised redirect URIs, add the URI shown above, then create.
  6. Copy the Client ID and Client Secret into the fields here and hit Save.
